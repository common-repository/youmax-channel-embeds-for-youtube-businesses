Youmax is one of the first YouTube plugins created. Initially this was created for plain HTML websites. After recieving a lot of request from WordPress users, we have launched a free WordPress version here.

Install the plugin and only then check out the [PRO version of Youmax](https://codecanyon.net/item/youmax-wp-grow-your-youtube-and-vimeo-business/10065614?ref=codehandling).

Some top features are:

*   YouTube Channel Header + Subscribe button
*   Video Grid
*   Tabs for Video Sources
*   Video Sources inlcude - YouTube Uploads, Playlists, even Vimeo User Videos etc.
*   Load More button

== Installation ==

1. Go to WP Admin > Plugins > Add New and upload the instalable ZIP.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Admin > Youmax > Add New screen to create Youmax Instances
4. Edit Youmax instances whenever you need.
